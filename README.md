# Simple To-Do List WebApp

A lightweight, responsive To-Do List application built with pure HTML, CSS, and JavaScript. This project focuses on clean UI design and persistent data storage using the browser's local memory.

## Features

* **Task Creation**: Quickly add tasks using the input field.
* **Persistent Memory**: Uses `localStorage` so your tasks survive a page refresh!
* **Task Management**: Mark tasks as "Done" with a visual strikethrough or "Delete" them permanently.
* **Responsive Design**: A centered, flexbox-powered layout that looks great on different screen sizes.
* **Clean Aesthetics**: A soothing green and beige color palette with card-based task displays.

## Built With

* **HTML5**: Semantic structure.
* **CSS3**: Flexbox, Aspect-Ratio, and Box-Shadows.
* **JavaScript (Vanilla)**: DOM Manipulation and Local Storage API.
